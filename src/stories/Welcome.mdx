<Subtitle>A React component library built with TypeScript and design tokens</Subtitle>

## Installation

```bash
npm install @sapyhte/design-system
```

**Peer dependencies:** `react` ^18.3.1, `react-dom` ^18.3.1

## Quick Start

```tsx
import { Button, Badge, Stack } from "@sapyhte/design-system";
import "@sapyhte/design-system/styles/index.css";

function App() {
  return (
    <Stack spacing={4}>
      <Badge colorPalette="saphyte">New</Badge>
      <Button variant="solid" colorPalette="saphyte">
        Get Started
      </Button>
    </Stack>
  );
}
```

## Component Categories

- **Layout** — Stack, Card, Divider, Separator, Tabs, Accordion, Table, Stat, Steps
- **Forms & Inputs** — Button, IconButton, Input, Textarea, Select, Checkbox, Radio, RadioGroup, Switch, Slider, SegmentedControl
- **Overlays** — Modal, Drawer, Popover, Tooltip
- **Navigation** — Breadcrumb, Menu, Pagination
- **Feedback** — Alert, Progress, EmptyState, Spinner, Skeleton
- **Data Display** — Badge, Tag, Avatar, Kbd, Icon
